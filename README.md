# Emotivoid
본 프로젝트는 생체 데이터를 활용한 RPG로, 사용자의 생체 데이터를 기반으로 게임 캐릭터의 능력치를 조정합니다. 사용자는 운동을 통해 게임 캐릭터를 성장시키며, 현실과 게임이 결합된 몰입감과 재미를 경험할 수 있습니다
